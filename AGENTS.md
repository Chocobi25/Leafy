# leafy

친환경 여행 설계 서비스 "Leafy"의 백엔드 프로젝트.

---

## 기술 스택

- Java 21
- Spring Boot 3.4.5
- Spring Data JPA
- QueryDSL
- MySQL
- Redis (제거 예정)
- Swagger / OpenAPI
- Docker
- Monitoring : Prometheus, Grafana

---

## 작업 원칙
- 반드시 `ai/ARCHITECTURE.md`, `ai/RULES.md`, `ai/REFACTORING.md` 내용을 우선적으로 따른다.
- 작업 완료 후 반드시 `./gradlew compileJava` 또는 관련 테스트를 실행하여 정상 동작 여부를 확인한다.
